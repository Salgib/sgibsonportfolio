Tipr

Tipr was my capstone app for my time at The Iron Yard. Tipr is a mobile tipping app centered around street performers. I used Ember.js as the framework to build it along with the Stripe API for payment processing and Parse as my backend database. It also takes advantage of the Google Maps API to help users find performers and their locations. Username: guest Password: guest
